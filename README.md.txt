# 🚀 Handpick Tool for Blind People

## 🔹 Overview
An **assistive device** empowering visually impaired individuals with **object detection, face recognition, and OCR**, using **Raspberry Pi** and smart **audio guidance**.

## 🌟 Key Features
✅ **Object Detection (YOLOv3)** – Identifies obstacles in real time  
✅ **Face Recognition (KNN)** – Helps recognize familiar individuals  
✅ **OCR (Text-to-Speech)** – Reads printed text aloud  
✅ **Ultrasonic Navigation** – Detects obstacles and alerts users  
✅ **Audio Feedback** – Guides movement via Bluetooth headphones
